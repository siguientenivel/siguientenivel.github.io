Intro al Desarrollo Web

Avanza al Siguiente Nivel aprendiendo a desarrollar aplicaciones web de manera ágil.

¡Inscríbete ya!

>Temario:

Plasma tus ideas con un prototipo

Aprende a crear un prototipo rápidamente utilizando Bootstrap y JQuery. Además aprende a hacer que tu aplicación se vea bien con nuestro minicurso: “Diseño para no diseñadores en 10 minutos (diploma no incluido)”.

Dale alma a tu App

Crea un API que le de alma a tu aplicación utilizando Node.js y express.js. Y lo mejor de todo: sin archivos de configuración, sin wizards ni servidores que consumen 1GB de RAM.

Dale superpoderes a tu App

Haz que tus usuarios compartan tu aplicación en redes sociales y no pierdas tiempo configurando una base de datos, deja que Firebase se encargue de guardarlos.

Comparte tu creación con el mundo

Pon tu aplicación online utilizando únicamente git y los servicios de heroku. ¡Sí!, es verdad, no tienes que saber cómo administrar un servidor ni escribir archivos de configuración.

>¿A quién está dirigido?:

Este curso está dirigido a personas con conocimientos básicos de programación y con muchas ganas de aprender ya que veremos muchos temas en poco tiempo. Sin embargo, si te gusta programar, si te gusta aprender cosas nuevas y si estás dispuesto a dedicarte 100% a aprender desarrollo web durante 2 días ¡Este curso es para ti!

>Metodología:

50% teoría + 50% práctica

Durante el curso desarrollaremos un proyecto dividido en 4 fases, cada una relacionada con un módulo del curso.
